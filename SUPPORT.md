# LibeR support

LibeR is research software. LibeRator is not a medical device, and no LibeR
package is qualified for autonomous clinical decision-making or regulatory
submission without a separate validation programme.

Use GitHub Discussions for installation and usage questions. Use the structured
issue forms for reproducible defects, numerical disagreements, performance
problems, GUI problems, and focused feature proposals.

Before reporting a problem, run:

```r
LibeRation::liber_doctor()
LibeRation::liber_support_bundle()
```

The support bundle is deliberately redacted by default. Inspect it before
uploading. Do not share patient data, credentials, restricted publications,
private model code, or identifiable screenshots.

Only the latest consolidated compatibility set receives routine fixes. The
exact package versions and wire/workspace contracts are recorded in
`ecosystem.json`.
