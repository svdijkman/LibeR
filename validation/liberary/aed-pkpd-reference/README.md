# AED-PKPD reference corpus versions

Use `0.1.2` for the current appendix-derived silver corpus.

- `0.1.0` was the first conversion pilot and omitted one historical six-digit PMID.
- `0.1.1` added that PMID but predates the corrected proportional-error classifier.
- `0.1.2` contains all 182 keyed appendix model records and is the supported baseline.

Older directories remain only to demonstrate immutable versioning. They must not
be mixed with `0.1.2` in benchmark or training exports.
